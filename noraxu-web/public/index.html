PROJECTS

Game of Squares

This game is implemented in System Verilog. It is played on a 3x3 rectangular grid consisting of black cells. When a cell is selected, its adjacent cells and itself will be flipped to their opposite colours. The objective is to make all the cells white.

Face Generator

The purpose of the project is to generate human faces that do not exist in the training set. Four models are compared: An autoencoder, a variational autoencoder, a Deep Convolutional Generative Adversarial Network, and a Variational Autoencoder Generative Adversarial Network. All models are trained on a processed version of the publicly available LFW face dataset.

Music Simulator

This project is implemented in assembly. It has four modes of operation: musical instrument, player, recorder, and modifier mode.

Pipelined Processor

This is a high performance pipelined processor consisting of four stages: fetch, decode, execute and write back.
